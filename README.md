
# Watch Footy Clone

A sports streaming website built with Next.js, matching the design of WatchFooty.st.

## Features

- **Live Sports Listing**: Fetches live sports and matches from the WatchFooty API.
- **Match Details**: View match information and stream players.
- **Date Filtering**: Filter matches by Yesterday, Today, and Tomorrow.
- **Admin Panel**: A dashboard to view site statistics and manage content (mock).
- **Responsive Design**: Fully responsive dark-themed UI.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure

-   `src/app`: Next.js App Router pages.
-   `src/components`: Reusable UI components.
-   `src/lib`: API client and utilities.
-   `src/styles`: Global styles and CSS modules.

## API

This project uses the public WatchFooty API. No API key is required.
